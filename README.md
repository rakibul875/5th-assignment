    Core Features

    API Integration (fetches and displays data dynamically), Dynamic Card System (includes name, price, features, description, image & badge), Add to Cart (add items with one click), Remove from Cart (easily delete items), Total Price Calculation (auto sum of cart items), Cart Management (add/remove functionality)

    UI & Design

    Tailwind CSS (modern and responsive design), DaisyUI (pre-built UI components), Lucide Icons (clean and lightweight icons), Responsive Design (works on mobile, tablet, and desktop)

    Additional Features

     Fast Performance (optimized rendering), Real-time Updates (React state-based UI), Clean Code (beginner-friendly structure)


     Technologies Used

     React.js, Tailwind CSS, DaisyUI, Lucide React Icons, JavaScript (ES6+), API Integration
